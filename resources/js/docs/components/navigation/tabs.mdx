## Documentation

### Tabs
#### Props

| Prop           | Type                                                                      | Default | Description                        |
|----------------|---------------------------------------------------------------------------|---------|------------------------------------|
| variant        | <TypeInfo title="variant" types={['default', 'underlined', 'classic']} /> | default | The variant style for the tabs.    |
| tabClassName   | string                                                                    | ``      | Custom classes for all tabs.       |
| panelClassName | string                                                                    | ``      | Custom classes for all tab panels. |
