## Documentation

### Props

[React Aria Button](https://react-aria.adobe.com/Button#api)

| Prop             | Type                                                                                                         | Default   |
  |------------------|--------------------------------------------------------------------------------------------------------------|-----------|
| variant          |<TypeInfo title="variant" types={['default', 'secondary', 'outline', 'ghost', 'destructive', 'link', 'toolbar', 'toolbar-default']} />| `default` |
| size          |<TypeInfo title="size" types={['default', 'sm', 'lg', 'icon', 'icon-sm', 'icon-xs', 'auto', 'full']} />| `default` |
| title            | string                                                                                                       | ''        |
| icon             | HugeIcons or Lucide icon                                            | ''        |
| isLoading          |  `boolean`                                                                                                     |  `false`     |
| iconClassName    | string                                                                                                       | ''        |
| tooltip          | string                                                                                                       | ``        |
| tooltipPlacement | Placement                                                                                                    | `bottom`  |
