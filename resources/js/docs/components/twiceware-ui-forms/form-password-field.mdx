## Documentation
### Props

| Prop         | Type      | Default            | Descriptions                                                       |
  |--------------|-----------|--------------------|--------------------------------------------------------------------|
| autoComplete | `string`  | `current-password` | AutoComplete field for password managers                           |
| description  | `string`  | ``                 | Helper text displayed below the label                              |
| isRequired   | `boolean` | `false`            | Whether the text field is required (shows a red asterisk in label) |
| label        | `string`  | ``                 | Label text for the text field                                      |
| showHint     | `boolean` | `false`            | Shows password requirements over the field                         |
| showStrength | `boolean` | `false`            | Shows a strength indicator in the field                            |
