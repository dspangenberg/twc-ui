## Documentation
### Props

| Prop                          | Type                                                                                          | Default                                                                                          | Descriptions                                                                               |
|-------------------------------|-----------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| bodyClassName                  | `string`                                                                                     | ``                                                                                          | ClassName for DialogBody                                                        |
| className                  | `string`                                                                                     | ``                                                                                          | ClassName for DialogContent                                                      |
| confirmClose                  | `boolean`                                                                                     | `false`                                                                                          | Closing of dialog must be confirmed                                                        |
| confirmationTitle             | `string`                                                                                      | <LangInfo de="Änderungen verwerfen" en="Discard changes" />                                      | Confirmation message                                                                       |
| confirmationMessage           | `string`                                                                                      | <LangInfo en="Do you want to discard the changes?" de="Möchtest Du die Änderungen verwerfen?" /> | Confirmation message displayed in the dialog                                               |
| confirmationButtonTitle       | `string`                                                                                      | <LangInfo de="Verwerfen" en="Discard" />                                                         | Label of the confirm/discard button                                                        |
| confirmationCancelButtonTitle | `string`                                                                                      | <LangInfo de="Abbrechen" en="Cancel" />                                                          | Label of the cancel button                                                        |
| confirmationVariant           | <TypeInfo title="variant" types={['default', 'destructive']} />                               | `default`                                                                                        | Variant of the OK-button                                                                   |
| description                   | `ReactNodeOrString`                                                                           | ``                                                                                               | Description of the dialog                                                                  |
| footer                        | `React.ReactNode` \|`(renderProps: DialogRenderProps)`                                        | ``                                                                                               | Footer                                                                                     |
| footerClassName                  | `string`                                                                                     | ``                                                                                          | ClassName for DialogFooter                                                     |
| hideHeader                    | `boolean`                                                                                     | `false`                                                                                          | Should the dialog header be hidden                                                            |
| isDismissible                 | `boolean`                                                                                     | `true`                                                                                           | Should the dialog close on ESC, <br/>outside interaction or the close-button in the header |
| showDescription               | `boolean`                                                                                     | `false`                                                                                          | If false the description is only visible on screenreaders                                  |
| title <Required />            | `string`                                                                                      | ``                                                                                               | Dialog title                                                                               |
| width                         | <TypeInfo title="width" types={['default', 'md', 'lg', '2xl', '3xl', '4xl', '5xl', '6xl']} /> | `default` (w-xl)                                                                                 | Width of the dialog                                                                        |
