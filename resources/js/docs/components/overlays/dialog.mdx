## Documentation
### Props

| Prop                    | Type                                                                                          | Default                                 | Descriptions                                                                               |
  |-------------------------|-----------------------------------------------------------------------------------------------|-----------------------------------------|--------------------------------------------------------------------------------------------|
| confirmClose            | `boolean`                                                                                     | `false`                                 | Closing of dialog must be confirmed                                                        |
| confirmationTitle       | `string`                                                                                      | `Änderungen verwerfen`                  | Confirmation message                                                                       |
| confirmationMessage     | `string`                                                                                      | `Möchtest Du die Änderungen verwerfen?` | Label of the OK button                                                                     |
| confirmationButtonTitle | `string`                                                                                      | `Verwerfen`                             | Label of the cancel button                                                                 |
| confirmationVariant     | <TypeInfo title="variant" types={['default', 'destructive']} />                               | `default`                               | Variant of the OK-button                                                                   |
| description             | `ReactNodeOrString`                                                                           | ``                                      | Description of the dialog                                                                  |
| footer                  | `React.ReactNode` \|<br/>`(renderProps: DialogRenderProps)`                                   | ``                                      | Footer                                                                                     |
| hideHeader              | `boolean`                                                                                     | `false`                                 | Should the dialog header hidden                                                            |
| isDismissible           | `boolean`                                                                                     | `true`                                  | Should the dialog close on ESC, <br/>outside interaction or the close-button in the header |
| showDescription         | `boolean`                                                                                     | `false`                                 | If false the description is only visible on screenreaders                                  |
| title <Required />      | `string`                                                                                      | ``                                      | Dialog title                                                                               |
| width                   | <TypeInfo title="width" types={['default', 'md', 'lg', '2xl', '3xl', '4xl', '5xl', '6xl']} /> | `default` (w-xl)                        | Width of the dialog                                                                        |
