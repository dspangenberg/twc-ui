| confirmClose            | `boolean`                                                                                     | `false`                                 | Closing of dialog must be confirmed |
| confirmationTitle       | `string`                                                                                      | `Änderungen verwerfen`                  | Confirmation message                |
| confirmationMessage     | `string`                                                                                      | `Möchtest Du die Änderungen verwerfen?` | Label of the OK button              |
